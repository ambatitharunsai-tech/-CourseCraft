# CourseCraft

CourseCraft is an AI-powered curriculum generator that creates structured semester-wise learning paths and professional PDF outputs.

## Features
- AI-generated curriculum
- Duration-based semesters
- Structured JSON output
- Professional PDF export
- Web interface
- Duplicate course prevention

## Duration Logic
- 3 Months → Half Semester
- 6 Months → 1 Semester
- 1 Year → 2 Semesters

## Tech Stack
Python • Flask • Ollama • HTML • CSS • JavaScript
